Game Rules

Rule 1:

Passive Flag

The red team hack into the blue team's server, and make a occupation, then get the points.

  • Occupation: Send the following request to the Flag Listener Server http://flag_listener/PassiveFlag/flag=HackBy{TeamName}

  • Reward: 10 points / success requests

Rule 2:

King Of Hill

Each team has been asked to attack the castle servers. After sending continunous 10 times RequestForKey request, the server will return back the key.

    1. Send the following request within 1 second interval, once a team successfully get a new key, the old key will expire. http://flag_listener/KingOfHill/key=RequestForKeyBy{TeamName}

    2. Send the key to FlagListener server, then a team can get the point http://flag_listener/KingOfHill/key={your_key}

  • Reward: 10 points / currect key request.

  • Note: the key is a 20 digits combination of numbers and alphabeta.

  • Occupation: Send the following request to the Flag Listener Server http://flag_listener/PassiveFlag/flag=HackBy{TeamName}

  • Reward: 10 points / success requests

This is a project about a simple passive flag listener and a scoreboard for a competition.
